# 메인 페이지 작업 정리

다음 PR로 합치기 전에 한 번 봐주시면 감사하겠습니다!
우선 제가 세훈님 브랜치에서 수정한 파트 정리했습니다.

## 바뀐 것

1. 지도 색상을 파란색으로 바꾸고, variation을 7단계로 키워서 눈에 더 잘 들어오게 했습니다.

2. 우측 두 번째 차트를 "동반자 유형" 막대에서 **Top 10 국적별 동반자 유형 (100%) 스택바**로 교체됐습니다. 사용한 데이터는 파일 (`src/data/companion_2024_2025.csv`) 및 세훈님이 올려주신 구글드라이브 데이터셋에도 올려두었습니다. 설문이 다중응답이라 응답률 합이 100% 넘는 경우가 있어서(예: 일본 약 108.6%) 각 국가 내에서 정규화했고, 호버 패널에는 정규화 비율과 원본 응답률을 같이 보여줍니다.

3. 우측 패널 하단에 **추천 숙박 구성 카드**를 새로 넣었습니다. Top 3 외국인 국적과 동반자→방 매핑(친구→트윈, 배우자→더블 등)으로 4개 방 유형(싱글/더블/트윈/패밀리)의 추천 비율을 자동 산출하도록 했습니다. 특히 트윈, 더블, 패밀리 유형에 맞춰서 각 동반자 유형도 색상을 비슷하게 묶어서 표현했습니다.

## 인터랙션

차트끼리 좀 더 연결되게 만들었습니다.

- 막대에 호버하면 그 국가만 양쪽 차트에서 동시에 강조
- 막대 클릭 = 범례 클릭: 그 카테고리로 정렬 
- 정렬되면 그 segment가 stack 첫 위치로 이동해서 모든 막대가 같은 baseline에서 시작 → 길이 직접 비교 가능 (dancing stack 아이디어 활용)
- 추천 카드의 방 유형 행을 클릭하면 그 방에 매핑된 카테고리들 합산 기준으로 정렬 (예: 패밀리 → 자녀+그외 가족+부모님)
- 정렬 활성 시 헤더에 칩 (`자녀 ↓ ✕`). ✕로 해제 가능

호버 정보 패널은 차트 위가 아니라 좌측 Top 10 영역 위에 띄웠어요.

## 색상

앞서 언급했듯 동반자 카테고리 색을 추천 카드의 방 유형 색과 같은 계열로 묶으면서, 탑10 그래프의 기존 푸른색을 회색으로 변경했습니다. 해당 그래프에서 푸른색으로 강조될 필요가 없을 것 같아서요!

- 친구·직장동료 = 푸른 계열 → 트윈
- 배우자/파트너 = 초록 → 더블
- 자녀·그외 가족·부모님 = 핑크 농도순(옅음→진함) → 패밀리
- 기타 = 회색 → 싱글

차트만 봐도 어느 동반자가 어느 방으로 매핑되는지 보이게 의도했습니다. 
Legend·stack·호버 패널 순서도 이 그룹에 맞춰 정렬돼 있어요.
